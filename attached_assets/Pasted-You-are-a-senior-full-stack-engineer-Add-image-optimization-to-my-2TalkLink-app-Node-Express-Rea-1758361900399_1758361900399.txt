You are a senior full-stack engineer.
Add image optimization to my 2TalkLink app (Node/Express, React/Tailwind, Supabase).

Goal:
- On upload, accept images (jpg/jpeg/png/gif/webp/avif) up to 2MB.
- Generate 3 WebP variants preserving aspect ratio (no enlargement):
  - thumb_200.webp     // max width/height 200
  - card_430.webp      // max width/height 430  (use on e-card, card width is 430px)
  - large_1200.webp    // max width/height 1200
- Store in Supabase Storage and return public URLs.
- PDFs pass-through (no transform).

Server:
- Deps: multer (memory), file-type (from buffer), sharp, @supabase/supabase-js, zod, express-rate-limit.
- ENV:
  SUPABASE_URL
  SUPABASE_SERVICE_ROLE_KEY
  SUPABASE_BUCKET=public-uploads
  ASSET_BASE_URL=https://<project>.supabase.co/storage/v1/object/public/public-uploads
  MAX_UPLOAD_BYTES=2097152
- Path pattern: user_{userId}/{YYYY}/{MM}/{DD}/{basename}/
- Upload route: POST /api/media/upload  (auth required)
  - Validate size ≤2MB and allowed mime
  - For images:
    * Save original (optional, as original.<ext>)
    * Create 3 buffers with sharp:
        const base = sharp(buffer).rotate();
        const v200 = base.clone().resize({ width: 200, height: 200, fit: "inside", withoutEnlargement: true }).webp({ quality: 80 });
        const v430 = base.clone().resize({ width: 430, height: 430, fit: "inside", withoutEnlargement: true }).webp({ quality: 80 });
        const v1200= base.clone().resize({ width: 1200,height: 1200,fit: "inside", withoutEnlargement: true }).webp({ quality: 78 });
    * Upload to bucket as:
        thumb_200.webp      (image/webp)
        card_430.webp       (image/webp)
        large_1200.webp     (image/webp)
      (and optionally original.<ext> with the detected content-type)
  - For PDFs: upload as-is with application/pdf.
  - Return JSON:
    {
      ok: true,
      type: "image" | "pdf",
      storagePath: "user_.../basename/",
      variants: {
        thumb_200_webp:  "<ASSET_BASE_URL>/.../thumb_200.webp",
        card_430_webp:   "<ASSET_BASE_URL>/.../card_430.webp",
        large_1200_webp: "<ASSET_BASE_URL>/.../large_1200.webp",
        original:        "<ASSET_BASE_URL>/.../original.<ext>" // if stored
      },
      width: <naturalWidth>, height: <naturalHeight>
    }
- Headers on JSON: X-Content-Type-Options: nosniff
