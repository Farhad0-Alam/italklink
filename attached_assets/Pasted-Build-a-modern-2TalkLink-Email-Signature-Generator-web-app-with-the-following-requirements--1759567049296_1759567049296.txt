Build a modern **2TalkLink Email Signature Generator** web app with the following requirements:

### Core Features
1. **Multiple Signature Templates**
   - Simple clean template (minimal, only profile + contact)
   - Advanced template (with logo, social icons, and optional CTA button)
   - Premium template (banner + call-to-action buttons like "Book a Consultation")

2. **User Input Form**
   - Fields: Name, Title, Company, Cell, Phone, Email, Website, Address
   - Upload option for profile photo & company logo
   - Add/remove social links (Facebook, LinkedIn, Instagram, WhatsApp, YouTube, TikTok, etc.)
   - Choose brand colors (primary & secondary) with a color picker
   - Toggle extra features (confidentiality note, CTA button, banner, etc.)

3. **Live Preview**
   - Instant preview on the right side as user types
   - Real email signature style with inline CSS
   - Responsive layout (desktop + mobile preview)

4. **Export Options**
   - **Copy HTML** button (one-click copy full inline CSS code)
   - **Download .HTML file** option
   - Shareable link to generated signature

### UI / UX Design
- **Modern & Simple:** clean white background, card-based layout
- **Branding:** Use 2TalkLink **orange (#ff6b00)** as primary highlight color
- **Typography:** Use **"Inter" or "Poppins"** font for modern professional look
- **Easy Controls:** Big input fields, toggle switches, and live preview side-by-side
- **Minimal Steps:** User can finish in under 2 minutes

### Technical Notes
- Frontend: React 19 + TailwindCSS (clean UI, responsive grid)
- Components: Shadcn/ui for buttons, modals, inputs
- Inline CSS generator (all email signatures must be generated with inline CSS for maximum email client support: Gmail, Outlook, Yahoo, Apple Mail)
- Store template structure in JSON for easy scalability
- Optimize for **non-technical users** (no coding needed)

### Example User Flow
1. User opens the generator → sees simple form
2. Fills in name, phone, email, and uploads photo/logo
3. Selects template (simple / advanced / premium)
4. Preview updates live
5. Clicks **"Copy HTML"** → ready to paste into Gmail/Outlook
6. (Optional) Downloads `.html` file

---
Goal: Deliver a **modern, easy, no-complexity** email signature generator that feels professional and fun to use, just like Canva-style editing but simpler.
