

Update and complete the 2TalkLink SaaS Admin Dashboard “Add New Plan” module.


🎯 Objective:
Build or update the full “Add New Plan” page in the Admin Dashboard that allows the admin to create or edit subscription plans, define pricing, and control which features are available for each plan.

---

### 📋 Form Fields & Descriptions (include these exactly)

| Field | Type | Example | Description |
|--------|------|----------|-------------|
| Plan Name | text | Pro Plan | The display name of the plan shown on the pricing page. |
| Plan Type | dropdown | Paid / Free | Defines whether the plan is free or paid. |
| Frequency | dropdown | Monthly / Yearly | Determines billing interval for the plan. |
| Currency | dropdown | USD | Default currency (USD or other supported). |
| Price | numeric | 12 | Monthly price or auto-calculated annual price (based on discount %). |
| Discount (%) | numeric | 20 | Optional — used for yearly plans to calculate discounted annual pricing. |
| Card Limit | numeric | 3 | Number of digital cards included in this plan. |
| Trial Days | numeric | 7 | Number of free trial days before billing starts. |
| Description | textarea | For creators and solopreneurs looking to grow and monetize | Short description of plan benefits. |
| Base Users Included | numeric | 1 | Number of users/cards included in base price. |
| Price Per Additional User | numeric | 0 | Extra charge (USD) per additional user/card. |
| Setup Fee | numeric | 0 | One-time setup cost (optional). |
| Maximum Users | numeric / text | Unlimited | Maximum number of users or cards allowed. |

---

### 🧮 Auto Calculation Logic
- If Frequency = “Yearly”, automatically calculate the annual price:
  `yearlyPrice = monthlyPrice * 12 * (1 - discount_percent / 100)`
- When the admin updates Discount (%) or switches between Monthly/Yearly, update the price in real time.
