You are a senior full-stack engineer.  
Enhance my 2TalkLink (Node/Express + React + Tailwind + Supabase) app to support uploading **HTML, images, and PDF** files that publish at top-level routes: https://2talklink.com/:slug

Allowed file types:
- .html
- .jpg
- .jpeg
- .png
- .gif
- .webp
- .avif
- .pdf

Constraints:
- Max size: 2 MB
- MIME must match extension
- One file per slug
- Public access via `/:slug`

DB schema (`public_html_pages`):
- id uuid pk
- user_id uuid fk
- slug text unique
- storage_path text not null
- title text null
- mime text not null
- is_public bool default true
- created_at timestamptz default now()
- updated_at timestamptz default now()

Storage:
- Supabase bucket: `public-uploads`
- Path: `user_{userId}/{slug}{ext}`

Reserved slugs (cannot be used):
[ "", "api", "auth", "admin", "builder", "dashboard", "login", "register",
  "logout", "static", "assets", "public", "favicon.ico", "robots.txt", "sitemap.xml",
  "_next", "card", "cards", "user", "users", "settings", "docs", "pricing",
  "health", "status", "webhook", "hooks", "oauth", "pay", "stripe", "paypal" ]

Backend (Express):
1. API endpoints (auth required):
   - POST /api/html-pages  
     * multipart: file, slug, title?  
     * Validate slug regex `^[a-z0-9-]{3,80}$`  
     * Reject if reserved or duplicate slug  
     * Validate file type & size (≤2MB)  
     * Upload to Supabase, insert DB row, return { url: `https://domain/${slug}` }
   - GET /api/html-pages (list user’s files)
   - PATCH /api/html-pages/:id (update title, toggle public, replace file with validation)
   - DELETE /api/html-pages/:id

2. Public route `GET /:slug`:
   - If slug is reserved → next()  
   - Lookup DB where slug = :slug and is_public = true  
   - If found:  
     * Option A: 301 redirect to Supabase public URL  
     * Option B: stream file with correct Content-Type  
       - .html → text/html; charset=utf-8  
       - .pdf → application/pdf  
       - .jpg/.jpeg → image/jpeg  
       - .png → image/png  
       - .gif → image/gif  
       - .webp → image/webp  
       - .avif → image/avif  
     * Always include headers:  
       X-Content-Type-Options: nosniff  
       Cache-Control: public, max-age=300  
   - Else → 404 page
