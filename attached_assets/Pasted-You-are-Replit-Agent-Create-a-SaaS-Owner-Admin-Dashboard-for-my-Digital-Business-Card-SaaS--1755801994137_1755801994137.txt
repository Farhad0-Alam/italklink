You are Replit Agent.  
Create a **SaaS Owner Admin Dashboard** for my Digital Business Card SaaS.  
This dashboard is only accessible to the **SaaS Owner** (root account) and any additional admins created by the owner.  

### Tech Stack
- Next.js 14 (App Router) + TypeScript
- TailwindCSS + shadcn/ui + lucide-react
- Prisma ORM with PostgreSQL
- NextAuth for auth
- Recharts/Chart.js for analytics

### Roles
- saas_owner (root, only 1 account, created at seed)
- admin (created only by saas_owner inside dashboard)
- org_admin (organization-level)
- member (normal user)

**Access rules:**
- `/admin/*` is restricted to role = `saas_owner` or `admin`.  
- `org_admin` and `member` can never access `/admin`.

### Database Models (Prisma)
- User: id, email, name, role (`saas_owner`, `admin`, `org_admin`, `member`), createdAt
- Organization: id, name, ownerId, plan, createdAt
- Membership: id, orgId, userId, role
- Template: id, scope (global/org), orgId?, name, schema JSON, previewImageUrl, createdBy, createdAt
- Card: id, orgId, userId, templateId, slug, data JSON, published
- Subscription: id, orgId, provider, status, currentPeriodEnd, planCode
- AnalyticsEvent: id, cardId, type(view|click|qr), ts, device, utm JSON
- AdminLog: id, actorId, action, targetType, targetId, ts

### SaaS Owner/Admin Dashboard Features
1. **Admin Management**
   - SaaS Owner can create new admins (email, name, password, role=admin)
   - SaaS Owner can delete or suspend admins
   - Admins cannot create new admins; only SaaS Owner can

2. **User Management**
   - List all users (search, filter by role, org)
   - Suspend or delete users
   - Reset user password

3. **Organization Management**
   - List all orgs with plan, owner, createdAt
   - Suspend/delete an org
   - Upgrade/downgrade org plan manually
   - View members of an org

4. **Template Management**
   - Create/edit/delete global templates (Minimal, Bold, Photo, etc.)
   - Approve/reject templates submitted by organizations

5. **Subscription Management**
   - View all active subscriptions (Stripe + PayPal)
   - See plan, provider, currentPeriodEnd
   - Cancel or adjust subscription manually

6. **Analytics Overview**
   - Show total cards, templates, users, orgs
   - Show monthly active users
   - Charts: daily views, QR scans, clicks across platform
   - Export CSV

### Routes
- `/admin` → dashboard overview (summary stats)
- `/admin/admins` → manage admins (only SaaS Owner can access this page)
- `/admin/users` → list/manage all users
- `/admin/orgs` → list/manage organizations
- `/admin/templates` → manage templates
- `/admin/subscriptions` → manage subscriptions
- `/admin/analytics` → charts and reports

### Security
- Middleware: protect `/admin/*` so only saas_owner or admin can enter
- SaaS Owner only can manage `/admin/admins`
- Every action stored in AdminLog (actor, action, target, timestamp)
